# 1  INTRODUCTION

## 1.1  Purpose of this guide

This User Guide describes the AI capabilities that have been recently added to the DISS platform for Schneider Electric GSC. It is written for shop-floor users — operators, team leaders, coaches — and for SMEs who configure DISS on a daily basis. The goal is to make every new feature operational from day one, without requiring previous experience with the underlying AI technology.

Each feature is described in a dedicated chapter following the same structure:
