# 1. Introduction

### Purpose of this guide

This User Guide describes the **AI capabilities** recently added to the **DISS platform for Schneider Electric GSC.** It is written for shop-floor users — operators, team leaders, coaches — and for SMEs who configure DISS daily. The goal is to make every new feature operational from day one, without requiring previous experience with the underlying AI technology.

### New AI features

| # | Feature                                       | Description                                                                              |
| - | --------------------------------------------- | ---------------------------------------------------------------------------------------- |
| 2 | **AI-Assisted 5W2H**   | Evaluates and scores Kata problem statement quality across 7 dimensions                  |
| 3 | **Historical Lookup** | Finds similar closed Katas and selectively copies their actions to the current one       |
| 4 | **DeepFind**                   | AI chatbot that answers natural-language questions against a configured Knowledge Source |
| 5 | **MeetingSense**           | Records SIM meetings, transcribes audio, and proposes actions automatically              |
| 6 | **Meeting Insight**     | Analyses patterns and answers questions across multiple meeting transcripts              |
| 7 | **Actions in SIM**       | Creates, assigns, and tracks corrective actions directly on the SIM Board                |

### Guide structure

Each chapter follows a consistent structure:

* **Overview** — a description of the feature and its purpose.
* **When to use it** — the typical scenarios in which the feature adds value.
* **Prerequisites** — what needs to be configured or enabled before the feature can be used.
* **How to use it** — step-by-step instructions with screenshots.
* **Reading the result** — how to interpret the output.
* **Tips & known limits** — practical advice and current boundaries of the feature.
